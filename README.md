# Deep Q Network Test
Somewhat clone version of Deep Q Learning Reinforcement Learning demo in ConvNetJS project

<img src="https://raw.githubusercontent.com/Chachay/DeepQNetworkTest/master/demo.png">

## Requirement
- Python 2.7 32bit
- Chainer 1.7.2
- numpy 1.11.0
- wxPython3.0-win32-py27

## Usage
```
python DQN001.py
```
